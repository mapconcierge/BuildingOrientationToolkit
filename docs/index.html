Building Shape Analysis Toolkit

Upload a GeoJSON file of building footprints (EPSG:4326) to compute area, roundness, centroid coordinates, and major axis orientation.

Global Statistics

How it works

  • Area is calculated on the ellipsoid and reported in square meters.
  • Roundness is computed as 4π × area / perimeter² where 1.0 represents a perfect circle.
  • Major axis orientation is derived from principal component analysis in a local UTM projection and is reported as azimuth clockwise from North.
